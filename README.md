# Klogged .NET

## About
Klogged is a basic logging library written in C# .NET Core 8.

## Features
- Different logger types
  - [x] Console logging 
  - [ ] File logging
- Logging options
  - [x] Colors in the console
  - [x] Log levels
  - Message formatting
    - [x] Custom color formatting of strings
    - [ ] Custom prefixes for messages
